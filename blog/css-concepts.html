Margins, Borders, and Padding

February 23, 2016


Every webpage contains HTML elements, and every HTML element is surrounded by an invisible box. These boxes contain the content you see, such as images and words, surrounded by attributes such as padding, borders, and margins. These attributes, which can be modified using CSS, can be tricky to tell apart.


Margins

Margins refer to the empty space around an element. Larger margins mean more space between an element on the page and neighboring elements.


Borders

Borders are the edges of an element. Borders can be made visible with the 'border' CSS property.


Padding

Padding is the area of space between an element's content (text, picture, video), and its border. With CSS, padding can be adjusted relative to its border.
