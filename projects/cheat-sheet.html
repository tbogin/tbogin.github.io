Ted's Favorite Ruby Enumerables Cheat Sheet

Name: any?

Why use: to discover if an object contains some element

How to use: 


  ["cat","dog","horse"].any? {|word| word.length > 3} --> returns true

Name: count

Why use: to find the number of times item is repeated within an object

How to use:


  [2,4,4,5].count{|num| num % 2 == 0} --> returns 3

Name: drop_while

Why use: to remove specified elements from an object

How to use:


  [1,2,6,7].drop_while{|i| i > 5} --> returns [1,2]

Name: each

Why use: iterates through an object, and manipulates object's items to your specifications

How to use:


  [2,4,5].each{|num| num * 2} --> doubles each array item

Name: each_slice

Why use: manipulate a set number of elements within an object

How to use:


  new_array = []
  [1,2,3,4,5,6].each_slice do |num|
    new_array.push(num) if num % 2 == 0
  end --> pushes even numbers into new array

Name: inject

Why use: great for adding, multiplying, etc... elements within an object

How to use:


  [1,2,3].inject{|result, number| result + number} --> returns sum of array: 6

Name: map

Why use: similar to each, except returns results within an array

How to use:


  [1,2,3].map{|i| i*2} --> returns [2,4,6]
